Project Overview

About the Project

This project was the final individual assignment for the UAS Electric Machine Control course at Sekolah Vokasi Universitas Gadjah Mada. It involved designing and implementing a 3-Phase Inverter Circuit with 3-Phase Motor Control that produces a perfect sinusoidal wave output.

Using Proteus for circuit simulation and Arduino IDE for programming the Arduino Uno microcontroller, the system was designed to ensure precise motor control with smooth sinusoidal output. The project provided in-depth exploration into motor control systems and inverter circuit design, allowing for the practical application of theoretical electrical engineering knowledge.

Throughout this project, valuable experience was gained in circuit design, simulation, and embedded programming. The hands-on nature of the work fostered a deeper understanding of the complexities of electrical machine control.
